
## Instalacion

```bash
$ npm install
```

## Ejecutar la aplicacion del lado del servidor

```bash
# Desarrollador
$ npm run start

# De modo que actulice cualquier cambio en el codigo
$ npm run start:dev

# Modo produccion
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
